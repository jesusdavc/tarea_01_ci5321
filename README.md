# Intrucciones
El siguiente repositorio contiene el código del ejemplo para la tarea 1 realizado en threeJS, TWGL, WebGL2. Realizado por Jesús Cuéllar. 

## Ejecución
### Descarga
Descargar ya sea usando descarga directa o por terminal:
* HTTPS:
```bash
git clone https://github.com/jesusdavc/tarea_01_ci5321.git
```
* SSH
```bash
git clone git@github.com:jesusdavc/tarea_01_ci5321.git
```
### ThreeJS
* Abrir la carpeta  ```threeJS ```
```bash
cd tarea_01_ci5321/threeJS/
```
* Ejecutar 
```bash
npm install
```
* Luego 
```bash

npx vite
```
* Contemplar
### TWGL
* Abrir la carpeta  ```TWGL```
```bash
cd tarea_01_ci5321/TWGL/
```
* Abrir ``` index.html``` en el navegador o si se usa VisualStudioCode desde la extensión LiveServer o alguna similar. 

* Contemplar.

### WebGL2
* Abrir la carpeta  ```webgl2```
```bash
cd tarea_01_ci5321/webgl2/
```
* Abrir ``` index.html``` en el navegador o si se usa VisualStudioCode desde la extensión LiveServer o alguna similar. 
* Contemplar.
